What Seismic Is

Seismic is a coding-activity tracking service. Through an editor extension (for Visual Studio Code and compatible editors), Seismic records how much time you spend coding, which projects and languages you work in, and related metadata. This data powers:

  • Dashboard — a personal view of your coding activity, time spent, and progress over time.
  • Goals — optional targets you can set for yourself (for example, a weekly coding-time goal), so you can track your own progress.
  • Leaderboard — an optional public or semi-public ranking of coding activity among users who choose to participate.
  • Profile — a page summarizing your activity, which you can make public or private.
  • API Key — a personal key that connects your editor extension to your Seismic account.
