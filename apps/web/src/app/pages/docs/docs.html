VS Code setup

  1. Open the Extensions panel (Ctrl+Shift+X / Cmd+Shift+X) and install the Seismic extension.
  2. Open the Command Palette (Ctrl+Shift+P / Cmd+Shift+P) and run Seismic: Set API Key.
  3. Paste in your API key from the box below.
  4. That's it — the status bar will show your tracked time once your first heartbeat goes through.

By default the extension talks to Seismic's production API — you only need to touch the API URL setting if you're running your own instance.

{{ selectedEditor().name }} setup

One plugin covers every JetBrains IDE — IntelliJ IDEA, PyCharm, WebStorm, GoLand, Rider, CLion, RubyMine, PhpStorm, DataGrip, Android Studio, and RustRover all use the same steps below.

  1. Go to Settings → Plugins → Marketplace, search for Seismic, and install it.
  2. Restart the IDE.
  3. Open Tools → Seismic → Set API Key… and paste in your key.
  4. Tracking starts automatically — check the Seismic item in the status bar for today's time.
