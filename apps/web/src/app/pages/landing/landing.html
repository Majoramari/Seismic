Frequently Asked Questions

Is this free?

The project is open-source and can be self-hosted.

Can I import old data?

Yes. Seismic includes WakaTime import support.

Is my code private?

Seismic records activity metadata, not source code contents.

Can I self-host this?

Yes. The API, web app, and editor integrations live in this repository.

How do badges work?

Badges are awarded from activity milestones and can be hidden from your public profile.

Which editors are supported?

Seismic currently supports VS Code, JetBrains IDEs, and Neovim.

What happens when I am offline?

Editor integrations queue heartbeats and retry them later when the API is reachable.

Can I set coding goals?

Yes. Goals can target overall time, languages, or projects by day, week, or month.
